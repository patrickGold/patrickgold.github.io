Featured Project

FlorisBoard

An open-source keyboard for Android. Currently in alpha stage.

Features
  • Support for latin keyboard layouts, e.g. QWERTY, QWERTZ, AZERTY, etc.
  • Emoji / Emoticon / Kaomoji input
  • Subtypes (language / layout pairs)
  • Theme selection through presets or fully customizable
  • One-handed mode, key press sound / vibration, ...

Rust Project

SimPi

A tool for simulating wiringPi projects written for the Raspberry Pi on Windows and GNU/Linux.

Features
  • Allows you to build and run programs written with the wringPi library
  • Supports most of the original library functions (including interrupts)
  • Works on GNU/Linux and Windows
